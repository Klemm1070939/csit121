Lessons Learned

Completing this portfolio helped me put together all the small projects from the semester into a single site. I learned how to structure pages with header, main, aside, and footer sections, and how to keep the layout consistent across pages. Testing on both Chrome and Firefox showed small differences in spacing that I fixed with simple CSS rules and the included media query.

Overall, the course taught me the basics of building web pages — writing semantic HTML, styling with CSS, and using small bits of JavaScript where needed. I learned that validating files and testing on multiple devices helps catch small issues that would otherwise become problems after publishing.
